# 🤝 Contributing to TheRustyGoat

Thank you for wanting to help build the future of mobile computing!

TheRustyGoat is a community-driven open-source project: a minimal Rust thin-client OS where the phone hardware is just a beautiful interface and all the power lives in the cloud.

**Every contribution matters** — code, ideas, testing, documentation, hardware experiments, or just "this is cool" comments.

### How to Contribute

**1. Small & Easy Ways**
- Star the repo ⭐
- Open an issue with ideas, bugs, or "Help wanted"
- Improve documentation or README
- Test on QEMU or your hardware and report results

**2. Code Contributions**
- Fork the repo
- Create a branch (`git checkout -b feature/amazing-thing`)
- Make your changes
- Open a Pull Request with a clear description

We welcome:
- Touch/input handling
- Streaming decoder improvements
- Hardware ports (PinePhone, dev boards, etc.)
- Networking optimizations
- Cloud backend examples
- Documentation & examples

**3. Big Ideas**
Want to own a whole module? Comment on the Roadmap issue — we love ambitious co-pilots.

### Code Style & Rules
- Keep it simple and safe (Rust safety is the point)
- Add comments explaining why
- Test locally with QEMU if possible
- Be kind and constructive

### Questions?
- Open a Discussion or comment on any issue
- Tag @SierzArt if you want founder feedback

**This project started with one person having a vision and not feeling "lame" enough to stop.**

Now it’s yours too.
